# Security Policy

## Reporting a vulnerability

This repository is a public portfolio and should not contain secrets.

If you believe you found something sensitive here (credentials, tokens, internal URLs/IDs), please **do not open a public issue**.
Instead, contact me directly:

- Email: `vstahanov@gmail.com`

## Scope

- This repo contains documents (CV/one-pagers/articles) and does not run production services.
- The main risk is accidental publication of sensitive information.
